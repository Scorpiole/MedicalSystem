# 医疗信息管理系统

### 依赖库

- Matplotlib     3.5.1
- Openpyxl      3.0.9
- pyqt5             5.15.6
- pathlib           1.0.1
- torch               暂时尚未整合诊断功能，诊断功能需要此库
- torchvision  



## 使用

#### 1、打开程序

打开WindowsDesign目录下的RunMainMedicsystemWindows.py,运行main。

#### 2、设置路径

打开窗口以后，点击菜单栏的设置路径，设置好 excel文件路径 和 对应的影像 **文件夹路径** 

ps：影像文件夹 中 必须包含 M0 和 M1 两个文件夹
